# Mood-Analyzer
A simple Java utility tool to analyze user mood from text input. 

This is a basic Java console application that asks the user about their current mood and classifies it as:
 • Positive
 • Negative
 • Neutral

It demonstrates simple text analysis and is suitable for practicing Version Control and Release Management, as new features or improvements can easily create new versions of the tool.

Features
 • Basic mood analysis using keywords
 • Easily extendable with more moods or features

Purpose

Created as a university assignment to apply Software Configuration Management (SCM) concepts on a simple Java utility project.


Example

What is your current mood?
I am feeling great today

Analyzing your input...
Mood is Positive
